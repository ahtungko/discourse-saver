关于 Discourse Saver

Discourse Saver 是一个 Chrome 浏览器扩展/油猴脚本，用于将 Discourse 论坛帖子一键保存到多个平台。支持 Obsidian、飞书多维表格、Notion Database、语雀知识库和思源笔记五种保存目标，可同时启用。

五平台保存

Obsidian、飞书、Notion、语雀、思源

评论获取

支持 API 获取完整评论

图片嵌入

支持 Base64 图片嵌入

自动检测

自动识别 Discourse 论坛

安装方式

方式一：Chrome 网上应用店（推荐）

在 Chrome 网上应用店搜索 "Discourse Saver" 或访问扩展页面直接安装。

方式二：手动安装

1. 从 GitHub Releases 下载最新版本的 ZIP 文件

2. 解压后在 Chrome 扩展管理页面（chrome://extensions/）启用"开发者模式"

3. 点击"加载已解压的扩展程序"，选择解压后的文件夹
